## Getting started

1. Clone repository:

   sh
   \$ git clone git://github.com/passeto/corebiz.git

2. create .env file

   ## add the next line:

   REACT_APP_API=https://corebiz-test.herokuapp.com/api/v1/

3. Go to project folder:

   sh
   \$ cd corebiz

4. Install all dependences:

   sh
   \$ npm install or yarn

5. Finally run:

   sh
   \$ npx start or yarn start

   Run: localhost:3000
